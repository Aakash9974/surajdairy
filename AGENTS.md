# Suraj Dairy Sales

Sales/ledger PWA for a small dairy. **Before doing anything, read `docs/PROGRESS.md`**
— it states the current phase and next step. Then see `docs/ROADMAP.md` (phase
checklists) and `docs/ARCHITECTURE.md` (layout & patterns). Update `docs/PROGRESS.md`
and tick `docs/ROADMAP.md` boxes at the end of each working session.

Stack: Next.js 16 (App Router) + React 19 + Tailwind 4 + Supabase. Money in INR.
